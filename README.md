# TPSai Analysis

Analysis code and reports for the development and validation of the Trust in Psychological Support from AI (TPSai) scale.

## Reports

| Report | Survey | Description |
|--------|--------|-------------|
| [Expert Evaluation](https://ricvolpe.github.io/tpsai-analysis/1_expert-evaluation.html) |  | Expert evaluation of item pool via inter-rater agreement |
| [Item Reduction](https://ricvolpe.github.io/tpsai-analysis/2_item-reduction.html) |  | Identifying items to drop or merge through quantitative pilot |
| [EFA](https://ricvolpe.github.io/tpsai-analysis/3_EFA.html) | | Exploratory factor analysis of each TPSai subscale using polychoric correlations and parallel analysis |
